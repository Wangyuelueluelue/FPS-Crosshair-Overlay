FPS 准星大师 (FPS Crosshair Overlay)

🎯 一个轻量级、无边框、始终置顶的 FPS 游戏屏幕准星辅助工具。使用 Python 和 Tkinter 开发。

✨ 核心功能

始终置顶：无论游戏如何切换，准星始终锁定在屏幕最上层。

鼠标穿透：点击准星中心完全不会导致游戏失去焦点。

透明背景：完美融合游戏画面，无黑框遮挡。

GUI 控制面板：提供可视化界面，一键切换准星颜色，一键安全退出。

🚀 如何使用 (玩家直接运行)

在本仓库右侧的 Releases 区域，下载最新的 Crosshair Overlay.exe 文件。

双击运行即可打开软件。

⚠️ 杀毒软件误报说明：由于本软件使用了 PyInstaller 打包且调用了 Windows 底层置顶/穿透 API，可能会被 Windows Defender 或其他杀毒软件误报为病毒。本软件完全开源，没有任何网络连接和恶意代码。请放心将其加入杀毒软件白名单。

游戏设置建议：请确保你的游戏运行在 “全屏无边框 (Borderless)” 或 “窗口化” 模式，独占全屏模式可能会遮挡准星。

💻 开发者指南 (从源码运行)

如果你想自己修改准星样式或功能，请按照以下步骤运行源码：

确保电脑已安装 Python。

下载本仓库的 Crosshair Overlay.py 文件。

运行源码：

python "Crosshair Overlay.py"
